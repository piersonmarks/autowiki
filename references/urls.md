# Canonical URLs

External services and tools referenced in the setup walkthrough. Verify the user gets to the right place — these are the official sources.

## Obsidian

- **Download:** https://obsidian.md
- **Help docs:** https://help.obsidian.md
- **Web Clipper browser extension:** https://obsidian.md/clipper
  - Works in Chrome, Firefox, Edge, Safari, Arc. The user installs it like any browser extension, then points it at their vault + a destination folder (`Raw/articles/`).

## Useful Obsidian plugins (optional)

None are required. Vanilla Obsidian is enough. If the user wants more:

- **Templater** — smart templating beyond the built-in Templates core plugin. https://silentvoid13.github.io/Templater/
- **Dataview** — query notes as if they were a database. Useful for dashboards over wiki pages. https://blacksmithgu.github.io/obsidian-dataview/
- **Advanced URI** — programmatic deep-linking into the vault from other apps. https://vinzent03.github.io/obsidian-advanced-uri/

Mention these only if the user asks "what else should I install?" — don't push them during the core setup flow.
